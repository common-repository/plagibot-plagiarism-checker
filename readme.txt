== Description ==

Plagiarism Checker by [Plagibot](https://plagibot.com) allows publishers to scan posts for plagiarism. Plagibot uses advanced AI and deep web-search technology to scan billions of web pages within seconds to give you the most accurate and up-to-date results. 

It also enables you to download a comprehensive report with results that publishers can easily share across teams for review and any follow-up action. 

It is well known that publishing duplicate or plagiarized content harms a website's reputation and can severely impact SEO. This lightweight plugin can help publishers to maintain the highest quality of plagiarism-free content on their websites. 

You can use our [Demo](https://plagibot.com) to try it out before installing.

== Installation ==

1. Install Plagibot from the WordPress.org plugin directory or by uploading the Plagibot plugin folder to your wp-content/plugins directory.
2. Sign up at [Plagibot.com](https://plagibot.com) and copy your API key
2. Use your API key to activate the plugin.
3. You will now see a "Plagiarism Check" button on your post edit page. Click on it to see the results.
